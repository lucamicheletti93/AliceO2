14 // Class to store the options for a Monte-Carlo prong history.
15 // First particle (index 0) in the std::vector corresponds to the most recent particle
16 // and continues with mother, grand-mother, ...
17 // Data members are: PDG codes, whether to check both charges, whether to exclude the specified PDG code,
18 // bit maps with a bit dedicated to each source (MCProng::Source), bit map on whether the specified source to be excluded,
19 // whether to use AND among all specified source requirements
20 
21 /* The PDG codes us the PYTHIA standard.
22 A few non-existent PYTHIA codes are used to select more than one PYTHIA code.
23 
24 0 - default, accepts all PYTHIA codes
25 100 - light unflavoured mesons in the code range 100-199
26 200 - --"-- 200-299
27 300 - strange mesons in the code range 300-399
28 400 - charmed mesons in the code range 400-499
29 401 - open charm mesons (all D and D* mesons) 400-439
30 402 - open charm mesons and baryons together 400-439, 4000-4399
31 403 - all (open- or hidden-) charm hadrons (mesons and baryons) in the range 400-499, 4000-4999 (! no psi' here)
32 404 - charged open charmed mesons w/o s-quark 410-419
33 405 - neutral open charmed mesons 420-429
34 406 - charged open charmed mesons with s-quark 430-439
35 500 - beauty mesons in the code range 500-599
36 501 - open beauty mesons 500-549
37 502 - open beauty mesons and baryons 500-549, 5000-5499
38 503 - all beauty hadrons 500-599, 5000-5999
39 504 - neutral open beauty mesons w/o s-quark 510-519
40 505 - charged open beauty mesons 520-529
41 506 - charged open beauty mesons with s-quark 530-539
42 902 - all open charm open beauty mesons+baryons 400-439, 500-549, 4000-4399, 5000-5499
43 903 - all hadrons in the code range 100-599, 1000-5999
44 1000 - light unflavoured baryons in the code range 1000-1999
45 2000 - --"-- 2000-2999
46 3000 - strange baryons in the code range 3000-3999
47 4000 - charmed baryons in the code range 4000-4999
48 4001 - open charm baryons 4000-4399
49 5000 - beauty baryons in the code range 5000-5999
50 5001 - open beauty baryons 5000-5499
51 */
